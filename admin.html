后台管理

说明

此后台页面已就绪。若要云端保存数据，请在 admin.js 中配置 Firebase。

前台已可直接使用，车辆数据来自 data/cars.json。

启用步骤（Firebase）

  1. 到 console.firebase.google.com 新建项目
  2. 开通 Authentication（Email/Password）与 Firestore
  3. 在“项目设置→应用”，复制网页 SDK 配置
  4. 把配置粘贴进 admin.js 顶部的 firebaseConfig
  5. 发布到 GitHub Pages，刷新 /admin.html 登录使用
